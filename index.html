📲 휴대폰에 설치

홈 화면에 추가하면 앱처럼 쓸 수 있어요.

설치 메뉴가 안 보일 때
  1. Chrome 앱으로 열었는지 확인 (카톡·슬랙 안에서는 설치 불가)
  2. 화면 맨 위 주소창을 한번 탭해서 메뉴가 나오게 하기
  3. 공유 버튼(↗ 모양) → 홈 화면에 추가
  4. 없으면 Chrome ⋮ (오른쪽 위 또는 아래) → 홈 화면에 추가

※ http:// 주소(IP)로 접속하면 「앱 설치」가 안 뜰 수 있어요. 공유 → 홈 화면에 추가는 됩니다.
